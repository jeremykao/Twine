Twine
=======

Project for Google Chrome Academy 2013 -- Twine is an app that allows anyone to discover people with the same interests and send personalized messages to them or invite them to an event.

## The Goods
- Google Users API
- Facebook FQL API
- HTML5 Geolocation
- jQuery
- Bootstrap
- Flat UI
